Guía 1

    La guia 1 se encarga de poder enseñarle a el alumno lo siguiente:

    1. Identificar la estructura básica de un documento web.

    2. Aplicar reglas de estilo CSS a los elementos HTML usando estilos embebidos y hojas de estilo externas.

    3. Utilizar los selectores CSS básicos y las propiedades CSS para la utilización de fuentes y textos.

Guía 2

    La guia 2 se encarga de poder enseñarle a el alumno lo siguiente:

    2. Conocer la utilidad de las pseudo-clases y pseudo-elementos CSS

    3. Crear páginas web utilizando el Modelo de Cajas, identificando las respectivas partes (margin, border, padding y content).

Guía 3

    La guia 3 se encarga de poder enseñarle a el alumno lo siguiente:

    1. Utilizar los diferentes tipos de posicionamiento CSS y diferenciarlos apropiadamente.

    2. Utilizar las propiedades CSS relacionada con la visualización de elementos.

    3. Maquetar sitios web con diseños fijos y fluidos.

Guía 4

    La guia 4 se encarga de poder enseñarle a el alumno lo siguiente:

    1. Crear y estilizar formularios utilizando la diversidad de controles de HTML5.

    2. Utilizar correctamente la meta-etiqueta viewport.

    3. Defina correctamente las media queries para aplicar reglas de estilo en función del ancho del viewport del navegador.

    4. Utilice los conocimientos adquiridos durante el transcurso de la asignatura para la construcción de páginas web responsivas.
